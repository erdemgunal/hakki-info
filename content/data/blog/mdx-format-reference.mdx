# MDX Blog Format Referansı

Blog yazıları `src/content/blog/` dizininde `.mdx` uzantısıyla tutulur.

---

## Frontmatter

Her `.mdx` dosyası YAML frontmatter ile başlar:

```mdx
---
title: "Fourier Dönüşümü ve Sinyal İşleme"
date: "2026-02-15"
excerpt: "Fourier dönüşümünün temel kavramları ve Python ile uygulamaları."
tags: ["math", "python", "signal-processing"]
---
```

| Alan      | Tip      | Zorunlu | Açıklama                                   |
|-----------|----------|---------|--------------------------------------------|
| `title`   | string   | ✓       | Başlık, `<h1>` olarak render edilir        |
| `date`    | YYYY-MM-DD | ✓    | Yayın tarihi                               |
| `excerpt` | string   | —       | Özet, liste sayfasında ve meta'da görünür  |
| `tags`    | string[] | —       | Etiketler (max 5 önerilir)                 |

## Örnek Tam Yazı

```mdx
---
title: "FFT ile Ses Analizi"
date: "2026-03-01"
excerpt: "Python ve NumPy kullanarak ses dosyasından frekans spektrumu çıkarma."
tags: ["python", "signal-processing", "numpy"]
---

Hızlı Fourier Dönüşümü (FFT), bir zaman-domain sinyalini frekans-domain'e çevirir.

## Temel Kavramlar

Fourier dönüşümü şu formülle tanımlanır:

$$
X(k) = \sum_{n=0}^{N-1} x(n)\, e^{-i 2\pi kn / N}
$$

Burada $N$ toplam örnek sayısını, $k$ ise frekans indeksini temsil eder.

## Python ile Uygulama

```python filename="fft_analysis.py"
import numpy as np
import matplotlib.pyplot as plt
from scipy.io import wavfile

# Ses dosyasını yükle
sample_rate, data = wavfile.read("ses.wav")

# FFT hesapla
fft_result = np.fft.fft(data)
frequencies = np.fft.fftfreq(len(data), 1 / sample_rate)

# Sadece pozitif frekanslar
positive_freq_idx = frequencies > 0
plt.plot(frequencies[positive_freq_idx], np.abs(fft_result[positive_freq_idx]))
plt.xlabel("Frekans (Hz)")
plt.ylabel("|X(f)|")
plt.show()
```

import { Callout } from '@/components/blog/MdxComponents'

<Callout type="info">
  Stereo ses dosyaları için önce `data = data[:, 0]` ile mono'ya çevirmeyi unutma.
</Callout>

## Sonuç

FFT, $O(n \log n)$ karmaşıklığıyla naif DFT'nin $O(n^2)$ karmaşıklığını geçer.
```
